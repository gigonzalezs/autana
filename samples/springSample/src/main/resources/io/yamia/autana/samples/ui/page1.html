What is functional programming?

 

Functional programming can help us create more robust, maintainable and easy-to-test software. Perhaps you have begun to hear about functional programming languages ​​such as Scala, Haskell or Lisp, but you may not yet know that Java in its version 8 allows you to use the power of functional programming without abandoning its object orientation.

 

C.A.R. Hoare says:

“There are two ways of constructing a software design: One way is to make it so simple that there are obviously no deficiencies, and the other way is to make it so complicated that there are no obvious deficiencies. The first method is far more difficult”.

 

With functional programming we are declaring the problem while writing its implementation in the code. All we have to do is take the requirements of the functionality we want to implement and write them. This way, if they ever change or if someone else has to look at our code, it will be very easy to know what they are doing now and, therefore, add the new specifications.
